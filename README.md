## Projeto criado com o intuito de aplicar conhecimentos em nest.js

## Instalação

```bash
$ npm install
```

## Rodar a api

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
